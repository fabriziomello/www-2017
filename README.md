# PGBR2017 Web Site

## How to Use

This website needs [hugo](http://gohugo.io/). Install it then run:

```bash
hugo serve
```

## How to Deploy

### Dependencies

Install gulp:

```bash
npm install --global gulp-cli
```

Install gulp modules:

```bash
npm install
```

### Deploying

Just run:

```bash
gulp deploy
```
